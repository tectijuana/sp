## Lineamientos de entrega

1. Toda entrega se realiza vía **GitHub** (pull request o repositorio propio enlazado), según indique la práctica.
2. Cada práctica incluye un `readme.md` con: objetivo, desarrollo, evidencias (capturas/simulaciones) y referencias en formato APA.
3. Si se usó IA en cualquier parte del trabajo, se adjunta la declaración de [`ANEXO.md`](../ANEXO.md).
4. El código sigue buenas prácticas: nombres claros, comentarios donde aportan, sin credenciales ni API keys en el repositorio.
