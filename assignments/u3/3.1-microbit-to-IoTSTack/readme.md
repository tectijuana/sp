## Script Python para Detecci√≥n Autom√°tica (Ejecutar en tu PC)
Si necesitan detectar el puerto program√°ticamente en tu computadora, usa este script (requiere `pyserial`):

### Instalaci√≥n de Dependencias
```bash
pip install pyserial
```
### Script
```python
import serial.tools.list_ports

# Identificar el micro:bit por su VID y PID
MICROBIT_VID = 0x0D28
MICROBIT_PID = 0x0204

def find_microbit_port():
    for port in serial.tools.list_ports.comports():
        if port.vid == MICROBIT_VID and port.pid == MICROBIT_PID:
            return port.device
    return None

if __name__ == "__main__":
    port = find_microbit_port()
    if port:
        print(f"Micro:bit encontrado en: {port}")
    else:
        print("Micro:bit no detectado.")
```

### Notas:
- El c√≥digo del micro:bit **es universal** y no depende del puerto.
- La detecci√≥n del puerto se realiza desde el lado de la computadora, no del micro:bit.
- Si usas editores como Mu o Thonny, detectar√°n autom√°ticamente el puerto.
